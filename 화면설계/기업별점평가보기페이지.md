
# 별점평가 글목록 페이지 이미지
![image](https://github.com/yyujjin/jjob-korea/assets/166809120/4a02169e-dd35-4291-9584-e4f4da7120f1)

# 구성설명
  ## 정렬 네비게이션
    - 좌측에 글을 정렬하는 네비게이션을 만든다
    - 하나는 평균 별점이 높은순으로 정렬하거나 낮은 순으로 출력해 글목록을 보여준다
    - 다른 하나는 해당 키워드를 가지고 있는 글들의 목록 순으로 출력한다.

  ## 기업 리뷰 리스트
  - 8개 정도의 리뷰 리스트를 보여준다.
  - 8개 이상의 글들은 번호를 이용해서 다음페이지로 이동해서 확인한다.
  - 작은 네모안에 기업이름과 이미지 별점의 평균을 표시한다.
  - 해당 네모를 클릭하면 기업별점평가내용을 확인하는 페이지로 이동한다.


# 기업별점평가내용 확인 페이지

#구성설명
  -기업의 이름 기업의 업종과 같은 정보표시
  -오른쪽에 평균 별점을 이미지와 함께 표시 ★★★★★
  - 가운데 내용으로는 여태까지 받은 키워드의 체크 수를 목록과 함께 표시한다.
  - 해당 기업에 관심을 가졌을 경우 상세 기업정보를 볼 수 있는 페이지로 이동하는 버튼
  - 다시 기업리뷰페이지 목록으로 이동하는 버튼
